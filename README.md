> ⚠️ **Requires OrbisManager** - This mod will not function without OrbisManager Core Mod.

InventorySync is a cross-server inventory synchronization addon for OrbisManager. It automatically transfers a player's full inventory including items, armor, and hotbar when they connect to a server on your network. There are no commands to learn; everything happens automatically in the background.

Inventory data is stored directly in OrbisManager's built-in Player Data system no database setup or additional configuration is required. Simply install the mod on every server where you want inventories to be synchronized and you are ready to go.

> **Note:** A player's inventory is tied to one active session at a time. If a player is connected to multiple servers simultaneously, only one inventory state will be active - preventing duplication or data conflicts across servers.

## Commands

No commands - Inventory is loaded and saved automatically when a player connects to or disconnects from a server.

## ✅ Plug & Play

No configuration needed. Install on your servers and OrbisManager handles the rest.
